Формат файла .inr
Каждая строка начинается с номера строки (первая строка имеет номер 100)
Значения в строке разделяются запятой
Значения, которые интерпретируются как текстовые строки, заключены в двойные кавычки

100
	Высота над уровнем моря (округляется до целых)
	Lock cylinders

101
	Путь к файлу
	Job Number (текст)
	Customer Name (текст)
	Address  (текст)
	Address (continued) (текст)
	City (текст)
	State/Province (текст)
	Postal Code (текст)
	Country (текст)
	Remarks (текст)

102
	const "T"
	система измерений: "METRIC" (СИ) или "ENG" (британская)
	Discharge Time (время в секундах)
	"N" (quantity of agent) \ "Y" (fixed nozzle)
	Number of Cylinders
	Cylinder Volume (fixed, eq 572 cu ft.)
	Pre-discharge Pipe Temperature

201
	Параметры помещения
	Под ним строка с перечнем форсунок (nozzle), которые располанаются в помещении, и количеством объемов n
	Под ним n строк с параметрами объемов (измерения параллелепипеда)


* В файле hpset.ini хранятся настройки программы:
система измерений: "M" (СИ) или "E" (британская)
путь к исполняемому файлу
результат отметки флажка "Check if autobalance is enabled" ("Y"/"N")
